4. 优先队列和 priority_queue

优先队列，意即优先级最高的的先出列。

它结合了队列和排序，不仅可以存储数据，还可以将这些数据按照设定的规则进行排序。每次的 push 和 pop 操作优先队列都会动态调整，把优先级最高的元素放在前面。

可以使用优先队列对数据进行排序：设定数据小的优先级高，把所有数 push 进入优先队列后一个个 top 出来，就得到了从小到大的序列。总复杂度是 O(nlog2n)。

5. 链表和 list

STL 的 list 的底层是双向链表，它的内存空间可以是不连续的，通过指针来进行数据的访问。它可以高效地在任意地方插入和删除。但是随机访问效率差。

6. set

set 就是集合。

STL 中的 set 使用二叉搜索树实现，集合中的每个元素只出现一次，并且是排好序的。访问元素的时间复杂度是 O(log2n)，非常高效。

7. map

map 是关联容器，它实现了从键 (key) 到值 (value) 的映射。它使用平衡二叉搜索树来存储和访问。查找的时间复杂度是 O(log2n)。

2. sort()

STL 中 sort() 底层是快排，复杂度是 O(nlog2n)。

当排序元素相等时且想保留原来的顺序时，可以使用 stable_sort。

partial_sort(): 局部排序。例如有 10 个数字，求最小的 5 个数，如果用 sort() 就需要全排序，再输出前5个。而使用 partial_sort 可以直接输出前 5 个。

3. next_permutation()

2. 搜索技术

1. 递归和排列

排列和组合是在暴力枚举的时候经常遇到的，它们的本质区别是：

  • 对排列来说，决策顺序对于结果有影响
  • 对组合来说，决策顺序对于结果无影响

一般有三种常见的情况：

  1. 打印 n 个数的全排列，共 个
  2. 打印 n 个数中任意 m 个数的全排列，共 个
  3. 打印 n 个数中任意 m 个数的组合，共 个

而递归是把大问题逐步缩小，直到变成最小的同类问题的过程。

例：使用递归求全排列

对于一个大小为 n 的序列，将序列的每个数分别与序列的第一个数交换一次位置，就可以得到 n 个序列。然后对于第2个数，我们也按照此方法处理，得到 (n-1) 个序列。依此类推，就得到了全排列。

2. BFS

深度优先搜索（DFS）和广度优先搜索（BFS）是基本的暴力技术，常用于解决图、树的遍历问题。

我们以老鼠走迷宫为例，来简单描述一下 DFS 和 BFS 模型。迷宫内部的路错综复杂，老鼠从入口进去怎么才能找到出口呢？

  • 一只老鼠走迷宫，它在每个路口都选择走右边（当然选择左边也可以），能走多远就走多远，直到碰壁无法继续往前走。然后回退一步，这一次走左边，接着继续往下走。用这个办法就能走遍所有的路，而且不会重复（这里规定回退不算重复走），这个思路就是 DFS
  • 一群老鼠走迷宫。假设老鼠是无限多的，这群老鼠进去后，在每个路口派出部分老鼠探索所有没走过的路。走某条路的老鼠，如果碰壁无法继续前行，就停下；如果到达的路口已经有其他老鼠探索过了，也停下。很显然，所有的道路都会走到，而且不会重复。这个思路就是 BFS。BFS 看起来像是“并行计算”，不过，由于程序是单机顺序运行的，所有可以把 BFS 看作是并行计算的模拟

在具体编程时，一般使用队列具体实现 BFS，使用递归实现 DFS。（DFS 也可使用递归来实现，栈和递归在算法思想上是一致的）
