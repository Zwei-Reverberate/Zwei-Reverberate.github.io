Ray Tracing

2022-10-28

1. 基本概念

  • Ray Casting

    Ray casting is the process of finding the closest, or sometimes just any, object along a ray.

    Ray Casting 是寻找光路上 (最近) 物体的过程。

    一条光线从摄像机出发，同屏幕上一个像素相交，直到它遇到最近的物体为止。若记它与物体的交点为 P，屏幕空间中的光源位置为 L，连接 L 和 P 来决定 P 是被照亮还是处于阴影之中。如果不在，就判定形成了一条有效的光路，由此计算与屏幕交点的着色。

Numeric Limit

2022-09-10

1. std::numeric_limits

C++ 标准库用 template numeric_limits 提供各种数值类型的极值。如需使用需要包含 < limits> 头文件。

实际上，numeric_limits 并不仅仅提供最大最小值：

The numeric_limits class template provides a standardized way to query various properties of arithmetic types.

而是提供了一种标准化的方式来查询数值类型的各种属性（摘自cppreference），而最大最小值只是诸多属性中的一种。

相比于传统 C 语言所使用的预处理常量（如 INT_MIN, INT_MAX, FLT_MAX…），新的极值概念有更好的类型安全性，因此提倡使用这种写法。
