位运算

2022-10-31

计算机中的所有数据都以二进制的方式存储，位运算直接对整数在内存中的二进制位进行操作，执行效率很高。（一般不对浮点数作位运算和移位运算，因为它的 bits 有特殊含义）

1. 位操作符

要在逻辑上使用位操作符，先将整数转换成二进制表示

符号 描述 规则
& 与 两位都为1时为1
\ 或 两位都为0时为0
^ 异或 两位相同为0，相异为1
~ 取反 各位0变1，1变0
<< 左移 向左进行移位操作，高位丢弃，低位补 0
>> 右移 向右进行移位操作，对无符号数，高位补 0。对于有符号数，高位补符号位（算术右移）

(*注：有的编译器对有符号数的右移是补0，即逻辑右移)

Ray Tracing

2022-10-28

1. 基本概念

  • Ray Casting

    Ray casting is the process of finding the closest, or sometimes just any, object along a ray.

    Ray Casting 是寻找光路上 (最近) 物体的过程。

    一条光线从摄像机出发，同屏幕上一个像素相交，直到它遇到最近的物体为止。若记它与物体的交点为 P，屏幕空间中的光源位置为 L，连接 L 和 P 来决定 P 是被照亮还是处于阴影之中。如果不在，就判定形成了一条有效的光路，由此计算与屏幕交点的着色。

1. C++ Input / Output Library

特点：

  • 类型安全

    IO operations are defined for each of the type. If IO operations are not defined for a particular type, compiler will generate an error.

  • 基于 streams of bytes 且设备无关
