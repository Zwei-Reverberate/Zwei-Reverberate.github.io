Ray Tracing

2022-10-28

1. 基本概念

  • Ray Casting

    Ray casting is the process of finding the closest, or sometimes just any, object along a ray.

    Ray Casting 是寻找光路上 (最近) 物体的过程。

    一条光线从摄像机出发，同屏幕上一个像素相交，直到它遇到最近的物体为止。若记它与物体的交点为 P，屏幕空间中的光源位置为 L，连接 L 和 P 来决定 P 是被照亮还是处于阴影之中。如果不在，就判定形成了一条有效的光路，由此计算与屏幕交点的着色。

  • Ray Tracing

    Ray Tracing 使用 Ray casting 递归地收集来自反射和折射物体的 light contributions。

    假设有一条光线从摄像机出发，同屏幕上的像素 A 相交，现在为了计算 A 的着色，就沿着光路做 Ray Casting。设光线遇到了镜面，且与镜面的交点为 P1，那么 P1 点也可能产生额外的反射和折射光线。假设其中一条反射光线发射之后又和空间内另一表面相交，交点为 P2, 那么从 P2 又可以产生额外的反射和折射光线。这个过程是不断递归的，不过递归通常有截止限制，例如最大 bounces 的次数。最终在计算 A 的着色的时候，需要将过程中产生的 P1, P2…Pn 全部和光源进行连线，每个点都对 A 的着色有影响，所以需要收集所有点的 light contributions。

  • 不同类型的 Ray Tracing

    • 在 Whitted Ray Tracing 中，物体表面是被视为完全光滑的，光源被视作是一个无穷小的点。

    • 在 Cook Ray Tracing 中，可以从 ray tree 的 nodes 中发射更多的光线以产生各种效果（不仅是光源可以发射更多的光线，过程中的每个反射点都可以发射更多的光线）。例如想象一下球形光源而不是点光源，我们可以向物体表面不同位置发射大量光线，在整合区域的 visibility 时，完全位于阴影中的区域是 Umbra，部分被遮挡的区域被称为 Penumbra。

      这种 Tracing 的缺点时从一点发射光线后，后续需要追踪的光线呈指数增长，消耗巨大。

      通过反射不同数量的光线以及不同的发射方向，可以模拟出不同的效果：

      Mirror：表面完全光滑，入射光线沿单一反向反射；

      Glossy：表面经过抛光处理，如黄铜等金属，在反射方向附近反射光线，外观有光泽

      Diffuse：材质为漫反射或哑光，入射光沿各个方向反射

    • 在 Path Tracing 中，光以能量的方式向各个方向辐射。 Cook Ray Tracing 的缺陷是要追踪的光线太多，而 Path Tracing 在每个点只发出一条反射和折射光线，以避免光线数量的激增。为了弥补采样太少的缺陷，每个像素都对多条光线进行追踪。所以 Cook Ray Tracing 在 ray tree nodes 中发出的光线最多，而 Path Tracing 在初始时发出的光线最多。

  • 渲染方程

    渲染方程能够求解出某个 Shading Point 的渲染。而光线从光源出发，经过不断的折射反射，最终到达某个 Shading Point，这个 Shading Point 直接连向屏幕上的一个像素和摄像机。那么屏幕上的像素渲染的结果就得到了，也就是光线追踪的结果，同时也是渲染方程的解。

    反过来看，我们从摄像机发出的 Ray，穿过屏幕上的某个像素之后，到达的也是某个 Shading Point，然后再逆向地递归去寻找光源。（对于打在光源以外的光路需要用其他方法停止递归）。

    光线追踪的渲染即求解渲染方程。

    而渲染方程是从能量的角度去描述光在空间中的传播和分布的。

    • 辐射度量学

      • 辐射能(Radiant Energy)，符号 Q，表示光穿过一个平面的光能，单位焦耳 (J)

      • 辐射通量(Radiant Flux(Power)), 符号 Φ，表示单位时间的光能

        $$ Φ = \frac{dQ}{dt} $$
        (亦称为 Power, 即功率)

      • 辐射强度(Radiant Intensity)，又叫光强度，符号 I，表示单位立体角上的辐射通量

        $$I(ω) = \frac{dΦ}{dω}$$
        （每单位立体角的 Power）

        立体角：一个立体角可以是球面面积 A 和半径 r2 之比，这个立体角的大小是和半径无关的。ω =

        $$\frac{A}{r^2}$$

        所以 Radiant Intensity 也可以理解成光源点朝某个立体角方向辐射 Power

      • 辐照度(Irradiance )，符号 E，表示单位面积上的辐射通量 （每单位面积的 Power）。

        $$E(x) = \frac{dΦ(x)}{dA}$$

        也可以理解成每单位面积接受的 Power

      • 辐射率（Radiance），又叫光亮度，符号 L，表示单位立体角、单位投影面积上的辐射通量（每单位投影面积的 Power）。

        $$L(x, ω ) = \frac{d^2Φ(x, ω)}{dωdAcosθ}$$

        也可以理解成，表面点上每单位面积接受 Power 后朝某个立体角方向反射的 Power

      Bidirectional Reflectance Distribution Function (BRDF) 表示的便在某个点 x 上，对于来自某个方向 ωi 的光照，可以朝某个方向 ωr 反射光的多少。（反射光 Radiance 和入射光 Irradiance 的比例）对应渲染方程中的 fr(x, ωi, ωr)。

      可以理解为在某一个特定的角度观看某个点时，各个方向的入射光对该点的最终光亮度产生的贡献比例。

          • 法线分布函数

            对应公式中的 D。

            法线分布函数 D(h) 描述了平面法线分布的概率，即具有正确朝向（能够将来自 ωi 方向的光反射往视点方向 ωo ）的微表面法线浓度。

            其物理意义是当微平面的法线分布比较集中（各法线朝向大致相同）时，那么物体表面材质会更容易表现出高光；当微平面的法线分布比较散开（各法线朝向差异比较大）时，那么物体表面材质将表现的非常 diffuse。

            法线分布函数相当于从微平面的角度去解释高光（Specular）部分。

              传统的 Blinn-Phong 模型通过 Specular = ksmax(0, n·h)p 确定 Specular 部分的运算，但是在辐射度量学的角度中渲染方程是一个余弦加权的半球积分，因此需要将其归一化：

              $$D(h) = \frac{p+2}{2π}(n·h)^p$$

              Beckmann NDF 是第一批微平面模型中使用的法线分布，也是Cook-Torrance BRDF在提出时选择的NDF。

              其中 α ∈ [0, 1] 表示为表面的粗糙程度。

              其中 α ∈ [0, 1] 表示为表面的粗糙程度。

            对应公式中的 G

            几何函数体现了光在物体微平面上反射时的损耗，一般指两种损耗：阴影（Shadowing）和遮蔽（Obstruction）。

            阴影（Shadowing）表示微平面对入射光的遮挡，即光源无法完全覆盖微平面；而遮蔽（Obstruction）表示微平面对出射光的遮挡，即微平面反射出的光不能完全到达摄像机观察点。

4. Sphere

1. 球与光线的交点

接下来往场景中添加物体。这里添加一个球体。

三维空间中球体的方程是 x2 + y2 + z2 = R2。任意给定一点 (x, y, z)，若在球上则有 x2 + y2 + z2 = R2，若在球内则有 x2 + y2 + z2 < R2，若在球内则有 x2 + y2 + z2 > R2。

若球心位于 C = (Cx, Cy, Cz)，那么对应的球体球体方程是：(x - Cx)2 + (y - Cy)2 + (z - Cz)2 = R2。

设点 P = (x, y, z)，那么 P 到球心 的向量为PC = (P - C)。假设 P 是球上一点，则应有：(P - C) · (P -C) = R2。

同时假设 P 位于光线上，带入光线方程 P(t) = A + tB，即 (A + tB - C) · (A + tB - C) = R2。

展开得到一个关于 t 的二次方程：(B ⋅ B)t2>+2B ⋅ (A − C)t + (A − C) ⋅ (A−C) − R2 = 0。

其判别式 Δ = 4(B ⋅ (A − C ))2 − 4(B ⋅ B)((A − C)⋅(A − C) − R2).

  • 若 Δ > 0，则方程有两不等实根，光线穿过球体
  • 若 Δ = 0，则方程有两相等实根，光线与球体相切
  • 若 Δ < 0，则方程无实根，光线和球体无交点

7. 漫反射材质

漫反射材质会往随机方向反射光线。

我们需要模拟这样的随机反射。假设一条光线命中了球体，命中的点为 P，那么可以定义一个与 P 相切的单位球，在这个球中随机找一个点 S，那么 (S - P) 就是我们想要的反射向量。记 N 为 P 点的单位法线，那么这个单位球的球心坐标可求得为 (P + N)。

10. Defocus Blur

散焦模糊，即景深。

我们在真实相机中散焦模糊的原因是因为它们需要一个大孔（而不仅仅是一个针孔）来收集光线。这会使所有东西都散焦。只要物体到相机的距离不等于焦距，就会出现模糊，光圈越大，采样射线的跨度越大，模糊效果越明显。

通常，所有场景光线都源自lookfrom 点。为了实现散焦模糊，生成随机场景光线，该光线源自以lookfrom 点为中心的圆盘内部。半径越大，散焦模糊越大。可以将我们的原始相机视为具有半径为零的散焦圆盘（根本没有模糊），因此所有光线都起源于圆盘中心（lookfrom）。
