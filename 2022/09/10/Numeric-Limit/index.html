Numeric Limit

2022-09-10

1. std::numeric_limits

C++ 标准库用 template numeric_limits 提供各种数值类型的极值。如需使用需要包含 < limits> 头文件。

实际上，numeric_limits 并不仅仅提供最大最小值：

The numeric_limits class template provides a standardized way to query various properties of arithmetic types.

而是提供了一种标准化的方式来查询数值类型的各种属性（摘自cppreference），而最大最小值只是诸多属性中的一种。

相比于传统 C 语言所使用的预处理常量（如 INT_MIN, INT_MAX, FLT_MAX…），新的极值概念有更好的类型安全性，因此提倡使用这种写法。

numeric_limits 在对所有数值类型提供一个通用的解决方案的同时，也为各个具体的类型提供了特化的版本。这也是使用 template 技术的一个要点（摘自《C++ 标准库 第2版》）：

  • 通用 template，为所有类型提供默认极值

    namespace std
{
    // general numeric limits as default for any type
    template <typename T>
    class numeric_limits
    {
      public:
        // by default no specialization for any type T exists
        static constexpr bool is_specialized = false;
        ...// other members are meaningless for the template
    }
}

  • 各具体类型的极值，由特化版本提供：

    namespace std
{
    // numeric limits for int
    template<> class numeric_limits<int>
    {
    public:
        static constexpr bool is_specialized = true;
        ...
    }
}

2. Template parameters

  • Member constants （皆为 static）

    Member Meaning
    is_specialized 类型是否有极值
    is_signed 类型是否有正负号
    is_integer 是否是整数类型
    is_exact 计算结果是否不产生舍入误差（此成员对所有整数类型而言均为 True）
    is_bounded 标识类型数值集是否有限（对所有内建类型而言均为 True）
    is_modulo 是否用模算数处理溢出（即两数相加，其结果可能因溢出而回绕为较小值）
    is_iec559 是否遵从 IEC559 及 IEEE754 标准
    radix 给定类型所使用的基数，即默认进制数。几乎总是2
    digits 使用 radix 的位数。（如 int 为31）
    digits10 10进制数字的个数（只有 is_bounded 成立才有意义）
    max_digits10 必要的10进制数位个数，确保不同的数值总是不同（since C++11）
    min_exponent 底 radix 的最小负整数指数
    max_exponent 底 radix 的最大负整数指数
    min_exponent 底10的最小负整数指数
    max_exponent 底10的最大负整数指数
    round_style 舍入风格
    has_infinity 是否有“正无穷大”的表述
    has_quiet_NaN 是否拥有 nonsignaling “Not a Number” 表述

  • Member Functions（皆为 static）

    Function Meaning
    min() 返回给定类型的最小值
    max() 返回给定类型的最大值
    lowest() 返回给定类型的最大负有限值
    epsilon() 返回1和最接近1的值之间的差距
    round_error() 最大舍入误差量度
    infinity() 返回“正无穷大”（如果有的话）
    quiet_NaN() 返回 quiet NaN value
    signaling_NaN() 返回 signaling NaN value
    denorm_min() 返回最小正值的 denormalized value

3. 使用示例

#include <limits>
#include <iostream>
#include <iomanip>

struct RowPrinter
{
    int m_left; // Left alignment
    int m_right; // Right alignment

    RowPrinter(int left, int right)
        :m_left(left), m_right(right)
    {
        // Print bool as 'true' or 'false'
        // Instead of 0 or 1
        std::cout << std::boolalpha;
    }

    template <typename A>
    auto printRow(const std::string& label, const A& value) const -> void
    {
        std::cout << std::setw(m_left) << label
                  << std::setw(m_right) << value << "\n";
    }
};

#define SHOW_INTEGER_LIMITS(numtype) \
    showNumericLimits<numtype>(#numtype)
#define SHOW_FLOAT_LIMITS(numtype) \
    showFloatPointLimits<numtype>(#numtype)


// Function to show the numeric limits
template <typename T>
void showNumericLimits(const std::string& name)
{
    RowPrinter rp{ 30, 25 };

    std::cout << "Numeric limits for type: " << name << "\n";
    std::cout << std::string(60, '-') << "\n";

    rp.printRow("Type:", name);
    rp.printRow("Is integer:", std::numeric_limits<T>::is_integer);
    rp.printRow("Is signed:", std::numeric_limits<T>::is_signed);
    rp.printRow("Number of digits:", std::numeric_limits<T>::digits);
    rp.printRow("Number of digits 10:", std::numeric_limits<T>::digits10);
    rp.printRow("Max Number of digits 10:", std::numeric_limits<T>::max_digits10);

    // RTTI - Run-Time Type Information
    if (typeid(T) == typeid(uint8_t) || typeid(T) == typeid(int8_t)
                                     || typeid(T) == typeid(bool)
                                     || typeid(T) == typeid(char)
                                     || typeid(T) == typeid(unsigned char))
    {
        // Min Abs - smallest positive value
        // for float point numbers
        rp.printRow("Min:", static_cast<int>(std::numeric_limits<T>::min()));
        // Smallest value (can be negative)
        rp.printRow("Lowest:", static_cast<int>(std::numeric_limits<T>::lowest()));
        // Largest value
        rp.printRow("Max:", static_cast<int>(std::numeric_limits<T>::max()));
    }
    else 
    {
        rp.printRow("Min:", std::numeric_limits<T>::min());
        rp.printRow("Lowest:", std::numeric_limits<T>::lowest());
        rp.printRow("Max:", std::numeric_limits<T>::max());
    }

    rp.printRow("Size in bytes:", sizeof(T));
    rp.printRow("Size in bits:", 8 * sizeof(T));
    std::cout << "\n";
}

// Function to show float points limits
template <class T>
void showFloatPointLimits(const std::string& name)
{
    RowPrinter rp{ 30, 25 };
    showNumericLimits<T>(name);
    rp.printRow("Epsilon:", std::numeric_limits<T>::epsilon());
    rp.printRow("Min exponent:", std::numeric_limits<T>::min_exponent10);
    rp.printRow("Max exponent:", std::numeric_limits<T>::max_exponent10);
}


int main()
{
    // Boolean type
    SHOW_INTEGER_LIMITS(bool);

    // Character types
    SHOW_INTEGER_LIMITS(char);
    SHOW_INTEGER_LIMITS(unsigned char);
    SHOW_INTEGER_LIMITS(wchar_t);

    // Standard integers in <cstdint>
    SHOW_INTEGER_LIMITS(int8_t);
    SHOW_INTEGER_LIMITS(uint8_t);
    SHOW_INTEGER_LIMITS(int16_t);
    SHOW_INTEGER_LIMITS(uint16_t);
    SHOW_INTEGER_LIMITS(int32_t);
    SHOW_INTEGER_LIMITS(uint32_t);
    SHOW_INTEGER_LIMITS(int64_t);
    SHOW_INTEGER_LIMITS(uint64_t);

    // Integer types
    SHOW_INTEGER_LIMITS(short);
    SHOW_INTEGER_LIMITS(unsigned short);
    SHOW_INTEGER_LIMITS(int);
    SHOW_INTEGER_LIMITS(unsigned int);
    SHOW_INTEGER_LIMITS(long);
    SHOW_INTEGER_LIMITS(unsigned long);
    SHOW_INTEGER_LIMITS(long long);
    SHOW_INTEGER_LIMITS(unsigned long long);

    // Floating types
    SHOW_FLOAT_LIMITS(float);
    SHOW_FLOAT_LIMITS(double);
    SHOW_FLOAT_LIMITS(long double);

    return 0;
}
